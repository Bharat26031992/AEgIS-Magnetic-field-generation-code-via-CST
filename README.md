# AEgIS-Magnetic-field-generation-code-via-CST

This project provides a high-performance surrogate model to replace computationally intensive Finite Element Method (FEM) simulations for a 23-coil electromagnetic system. By leveraging the principle of magnetic superposition, this MATLAB-based tool reduces simulation time from minutes (CST Studio Suite) to milliseconds, enabling real-time digital twin capabilities and rapid field optimization.

Key Features 
Real-Time Synthesis: Instantaneous calculation of magnetic flux density for any combination of 23 coil currents.Data Regularization: Automated scripts to interpolate scattered CST mesh data onto a unified Master Grid.Interactive GUI: A standalone MATLAB dashboard featuring:23-channel current control.Live 2D $B_{mag}$ contour mapping.1D axial $B_z$ profile visualization.Validated Precision: Initialized with verified experimental current values for 5T and 1T operation modes.

Mathematical Theory
The model operates on the principle of linear superposition. In a non-saturable system, the total magnetic flux density $B$ at any point $r$ is the sum of the fields generated by individual coils:$$B_{total}(r) = \sum_{i=1}^{n} I_i \cdot b_i(r)$$Where:$I_i$: The user-defined current for the $i^{th}$ coil.$b_i(r)$: The Basis Vector (unit field), calculated by normalizing a single CST simulation by its current $I_{sim,i}$:$$b_i(r) = \frac{B_{sim,i}(r)}{I_{sim,i}}$$.

Data Pre-processing:
Since CST uses an adaptive mesh, the exported $(x, y, z)$ coordinates vary between simulations. This tool includes a regularization pipeline:
Import: Parses raw ASCII data from CST.
Slicing: Extracts the $x = 0$ plane for 2D analysis.
Interpolation: Uses scattered Interpolant to map all basis fields onto a consistent Master Grid, ensuring matrix compatibility ($O(1)$ lookup time).

GUI Architecture
The app is built using MATLAB App Designer with a non-blocking callback architecture:

Input: 5-column grid for 23 independent current entries.

Logic: Vectorized matrix-vector multiplication for near-zero latency.

Output: Dynamic UI axes showing field homogeneity and axial profiles.
